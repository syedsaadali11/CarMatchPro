How Our Algorithm Works

A step-by-step breakdown of our recommendation process

1

Data Collection

Gather comprehensive vehicle data including specifications, pricing, and market information.

2

Data Preprocessing

Clean and normalize data, encode categorical variables, and scale numerical features.

3

Clustering

Apply K-Means clustering to segment vehicles into similar groups based on characteristics.

4

User Input Processing

Transform user preferences using the same preprocessing pipeline as training data.

5

Cluster Assignment

Determine which cluster the user's preferences belong to using the trained model.

6

Similarity Calculation

Calculate cosine similarity between user preferences and vehicles in the assigned cluster.

7

Ranking & Results

Rank vehicles by similarity score and return the top recommendations to the user.
